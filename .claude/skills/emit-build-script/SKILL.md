# Emit build_rocrate.py

This skill writes the **canonical script** that builds the RO-Crate. It's modeled on `fairscape-repos/kaggle_rocrate.py` — pure Python using `fairscape_models.*.model_validate({...})`, manual `@graph` assembly, write `ro-crate-metadata.json`. The script is meant to be re-runnable and human-editable; the wizard state file is just scaffolding.

## Before emitting

- Run `plausibility-check` first if it hasn't been run for this state. Block on resolved issues.
- Confirm with the user: "Ready to write `build_rocrate.py` and run it?"

## What gets inlined vs. computed at runtime

- **Inlined**: every single-entity record from `state.datasets` / `state.software` / `state.computations`, plus each bulk group's template + glob.
- **Computed at runtime**: bulk-group GUIDs (deterministic from filenames), per-file Dataset records expanded by `for path in sorted(Path(root).glob(...))` loops.

This means the script picks up new files matching a bulk glob without regeneration — which is what the user wants for ongoing acquisitions.

## Procedure

1. `Read` `.fairscape-wizard-state.json`.
2. Compose `build_rocrate.py` matching the **template** below. Fill in:
   - `CRATE_METADATA = {...}` from `state.crate_metadata` + `state.rai`.
   - `SINGLE_DATASETS = [...]` from `state.datasets` (one dict per entry, including `guid`, `name`, `author`, `description`, etc., plus a per-record `is_raw_input` flag — NOT emitted into the JSON, just used to skip back-wiring).
   - `BULK_GROUPS = [...]` from `state.bulk_groups`.
   - `SOFTWARE = [...]` from `state.software`.
   - `COMPUTATIONS = [...]` from `state.computations`, with `usedDataset` / `usedSoftware` / `generated` referencing GUIDs by string and `used_bulk_groups` / `generated_bulk_groups` referencing prefixes.
3. `Write` the file to `<project_root>/build_rocrate.py`.
4. Run it: `Bash python build_rocrate.py` from the project root.
5. Run `Bash fairscape rocrate validate <project_root>`.
6. Report. If validate fails, surface the message and route the user back to the relevant skill.

## Template

```python
#!/usr/bin/env python3
"""Build ro-crate-metadata.json from inline metadata + bulk file globs.

Generated by the FAIRSCAPE RO-Crate wizard. Edit freely — this script is the
canonical artifact. Re-running it overwrites ro-crate-metadata.json.
"""
from __future__ import annotations

import hashlib
import json
from datetime import date
from pathlib import Path

from fairscape_models.computation import Computation
from fairscape_models.dataset import Dataset
from fairscape_models.rocrate import (
    ROCrateMetadataElem,
    ROCrateMetadataFileElem,
    ROCrateV1_2,
)
from fairscape_models.software import Software

ROOT = Path(__file__).parent
NAAN = "59853"
DEFAULT_CONTEXT = {
    "@vocab": "https://schema.org/",
    "EVI": "https://w3id.org/EVI#",
}

CRATE_GUID = "<ark:59853/rocrate-...>"  # filled in at emit time

CRATE_METADATA = {
    "name": "...",
    "description": "...",
    "authors": ["..."],
    "keywords": [],
    "datePublished": "YYYY-MM-DD",
    "license": None,
    "version": "1.0",
    "associatedPublication": None,
    "rai": {
        "dataLimitations": None,
        "dataBiases": None,
        "dataCollection": None,
    },
}

SINGLE_DATASETS = [
    # one dict per state.datasets entry
    # {"guid": "ark:...", "name": "...", "author": "...", "description": "...",
    #  "datePublished": "...", "keywords": [...], "format": "...", "version": "1.0",
    #  "contentUrl": "file:///..."}
]

BULK_GROUPS = [
    # one dict per state.bulk_groups entry
    # {"guid_prefix": "dataset-microscopy-image",
    #  "glob": "raw/images/*.tif",
    #  "template": {"name_template": "Microscopy image {basename}",
    #               "description_template": "...", "author": "...",
    #               "keywords": [...], "format": "image/tiff", "version": "1.0",
    #               "datePublished": "..."},
    #  "overrides": {"raw/images/control_01.tif": {"description": "..."}}}
]

SOFTWARE = [
    # one dict per state.software entry
]

COMPUTATIONS = [
    # one dict per state.computations entry — includes used_bulk_groups / generated_bulk_groups
    # which the assembler expands into per-file GUIDs.
]


def slugify(name: str) -> str:
    import re
    s = name.lower().strip()
    s = re.sub(r"\s+", "-", s)
    s = s.replace(".", "-").replace("/", "-")
    s = re.sub(r"[^a-z0-9-]", "", s)
    s = re.sub(r"-+", "-", s).strip("-")
    return s or "entity"


def bulk_guid(prefix: str, basename: str) -> str:
    """Deterministic GUID for a bulk-expanded file: ark:NAAN/<prefix>-<slug(basename)>."""
    return f"ark:{NAAN}/{prefix}-{slugify(basename)}"


def expand_bulk_group(group: dict) -> list[Dataset]:
    """Expand one bulk group into a list of Dataset model instances."""
    out = []
    glob = group["glob"]
    template = group["template"]
    overrides = group.get("overrides", {})
    for path in sorted(ROOT.glob(glob)):
        if not path.is_file():
            continue
        rel = path.relative_to(ROOT).as_posix()
        basename = path.name
        stem = path.stem
        record = {
            "@id": bulk_guid(group["guid_prefix"], basename),
            "name": template["name_template"].format(basename=basename, stem=stem),
            "author": template["author"],
            "description": template["description_template"].format(basename=basename, stem=stem),
            "datePublished": template["datePublished"],
            "keywords": list(template.get("keywords", [])),
            "format": template["format"],
            "version": template["version"],
            "contentUrl": f"file:///{rel}",
        }
        # apply per-file overrides
        record.update(overrides.get(rel, {}))
        out.append(Dataset.model_validate(record))
    return out


def expand_bulk_group_guids(group: dict) -> list[str]:
    """GUID-only expansion for use in computation links."""
    return [
        bulk_guid(group["guid_prefix"], path.name)
        for path in sorted(ROOT.glob(group["glob"]))
        if path.is_file()
    ]


def build_single_dataset(d: dict) -> Dataset:
    return Dataset.model_validate({
        "@id": d["guid"],
        "name": d["name"],
        "author": d["author"],
        "description": d["description"],
        "datePublished": d["datePublished"],
        "keywords": d.get("keywords", []),
        "format": d["format"],
        "version": d["version"],
        "contentUrl": d["contentUrl"],
    })


def build_software(s: dict) -> Software:
    return Software.model_validate({
        "@id": s["guid"],
        "name": s["name"],
        "author": s["author"],
        "description": s["description"],
        "dateModified": s["dateModified"],
        "version": s["version"],
        "format": s["format"],
        "contentUrl": s["contentUrl"],
    })


def build_computation(c: dict, bulk_index: dict[str, dict]) -> Computation:
    used_dataset_ids = list(c.get("usedDataset", []))
    for prefix in c.get("used_bulk_groups", []):
        used_dataset_ids.extend(expand_bulk_group_guids(bulk_index[prefix]))
    generated_ids = list(c.get("generated", []))
    for prefix in c.get("generated_bulk_groups", []):
        generated_ids.extend(expand_bulk_group_guids(bulk_index[prefix]))
    return Computation.model_validate({
        "@id": c["guid"],
        "name": c["name"],
        "description": c["description"],
        "runBy": c["runBy"],
        "dateCreated": c["dateCreated"],
        "usedSoftware": [{"@id": x} for x in c.get("usedSoftware", [])],
        "usedDataset": [{"@id": x} for x in used_dataset_ids],
        "generated": [{"@id": x} for x in generated_ids],
    })


def main() -> Path:
    bulk_index = {g["guid_prefix"]: g for g in BULK_GROUPS}

    single_datasets = [build_single_dataset(d) for d in SINGLE_DATASETS]
    bulk_datasets: list[Dataset] = []
    for g in BULK_GROUPS:
        bulk_datasets.extend(expand_bulk_group(g))
    software_entities = [build_software(s) for s in SOFTWARE]
    computations = [build_computation(c, bulk_index) for c in COMPUTATIONS]

    all_entities = single_datasets + bulk_datasets + software_entities + computations
    has_part = [{"@id": e.guid} for e in all_entities]

    metadata_file_elem = ROCrateMetadataFileElem.model_validate({
        "@id": "ro-crate-metadata.json",
        "@type": "CreativeWork",
        "conformsTo": {"@id": "https://w3id.org/ro/crate/1.2"},
        "about": {"@id": CRATE_GUID},
    })

    crate_root = ROCrateMetadataElem.model_validate({
        "@id": CRATE_GUID,
        "@type": ["Dataset", "https://w3id.org/EVI#ROCrate"],
        "name": CRATE_METADATA["name"],
        "description": CRATE_METADATA["description"],
        "keywords": CRATE_METADATA.get("keywords", []),
        "version": CRATE_METADATA["version"],
        "author": ", ".join(CRATE_METADATA["authors"]) if CRATE_METADATA.get("authors") else "",
        "license": CRATE_METADATA.get("license"),
        "datePublished": CRATE_METADATA["datePublished"],
        "hasPart": has_part,
    })

    graph = [
        metadata_file_elem.model_dump(by_alias=True, mode="json"),
        crate_root.model_dump(by_alias=True, mode="json"),
    ]
    for e in all_entities:
        graph.append(e.model_dump(by_alias=True, mode="json"))

    rocrate = ROCrateV1_2.model_validate({"@context": DEFAULT_CONTEXT, "@graph": graph})
    out_file = ROOT / "ro-crate-metadata.json"
    with open(out_file, "w") as f:
        json.dump(rocrate.model_dump(by_alias=True, mode="json"), f, indent=2)
    print(f"Wrote {out_file}  ({len(graph)} graph nodes)")
    return out_file


if __name__ == "__main__":
    main()
```

## Filling in the template

For each section, format as a Python list of dicts using JSON-like syntax. Long string fields can use double-quoted strings; multi-line descriptions can use triple-quoted strings.

For `CRATE_GUID`, use `ark:59853/rocrate-<slug(crate_metadata.name)>-<squid>`. Generate a fresh squid via `python -c 'from fairscape_wizard import generate_guid; print(generate_guid("rocrate", "<name>"))'` — this becomes a constant in the file so re-running keeps the same crate GUID.

If `state.bulk_groups` is empty, leave `BULK_GROUPS = []` and the bulk loop simply produces no entities. Same for any other empty section.

## After writing the file

1. `Bash python <project_root>/build_rocrate.py` — confirm it runs.
2. `Bash fairscape rocrate validate <project_root>` — confirm schema validation passes.
3. Tell the user: `wrote build_rocrate.py and ro-crate-metadata.json. Validation passed.` (or surface the error).

## On validation failure

Don't panic-edit. Read the validator message, identify which field/entity is wrong, route the user to the responsible skill (`extract-crate-metadata` for top-level fields, `register-dataset` etc. for entity-level), apply their fix to state, and re-emit.

## On user-edited build_rocrate.py

If the user has edited the script and you need to regenerate, **stop and ask**. Don't silently overwrite their edits. Offer to: (a) emit to `build_rocrate.generated.py` instead, or (b) have them confirm overwrite.
